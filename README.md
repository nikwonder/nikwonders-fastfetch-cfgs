# how to install
you can either manually do this or clone this repo and use the `install.sh` given

## automatic
clone the repo:
```
git clone https://github.com/nikwonder/nikwonders-fastfetch-cfgs.git
```

open the directory:
```
cd nikwonders-fastfetch-cfgs
```

start the script:
```
./install.sh
```

## manually
download the config you want, then copy it into the `~/.config/fastfetch/` folder
if you want to use it permanently, rename the file to `config.jsonc`
if you don't, use `fastfetch -c [path to config here]`
